Two options - update or restore the entire device

WX station firmware

Firmware version: 20260629 for Hardware

Update via the EasyOTA web service

Download the two .bin files and upload them on the device web update OTA page http://[ip]:82/update in this order: first firmware, then filesystem.

Files for EasyOTA update:

Firmware 20260629.bin SPIFFS 20260629.bin

Complete recovery via USB

Use this path for device recovery. Open this page in Google Chrome or Microsoft Edge over https://, connect the WX station to your computer using USB, then start the automated install below.

  • Before recovery, back up your configuration from the Setup page http://[ip]:80/setup
  • After pressing the Connect button, select the USB-to-UART bridge device
  • Then select the Install WX Station Recovery option
  • After uploading, select the USB device again and the Logs & Console option, where after pressing Reset Device you will see the boot log, including the device's IP address

The installer flashes bootloader, partition table, boot_app0, firmware, and spiffs automatically.
